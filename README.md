KaiOs CLI to execute adb shell commands faster and send sms
### Features
+ reboot device
+ backup sms db
+ send sms from command line

![image-2](/images/image-1.png)


### Installation
+ copy sendsms to your phone adb push sendsms /system/bin/
+ make KaiOsCli.sh executable
